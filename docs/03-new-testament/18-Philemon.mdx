# 腓利门书

## 腓利门书 1 {#1}

### 问候

为基督耶稣被囚的<Pn>保罗</Pn>，同兄弟<Pn>提摩太</Pn>写信给我们所亲爱的同工<Pn>腓利门</Pn>，和妹子<Pn>亚腓亚</Pn>并与我们同当兵的<Pn>亚基布</Pn>，以及在你家的教会。愿恩惠、平安从　神我们的父和主耶稣基督归与你们！<br/>

### 腓利门的爱心和信心

我祷告的时候提到你，常为你感谢我的　神；因听说你的爱心并你向主耶稣和众圣徒的信心<Nt>或译：因听说你向主耶稣和众圣徒有爱心有信心</Nt>。愿你与人所同有的信心显出功效，使人知道你们各样善事都是为基督做的。兄弟啊，我为你的爱心，大有快乐，大得安慰，因众圣徒的心从你得了畅快。<br/>

### 保罗为阿尼西谋请求

我虽然靠着基督能放胆吩咐你合宜的事，然而像我这有年纪的<Pn>保罗</Pn>，现在又是为基督耶稣被囚的，宁可凭着爱心求你，就是为我在捆锁中所生的儿子<Pn>阿尼西谋</Pn><Nt>就是有益处的意思</Nt>求你。他从前与你没有益处，但如今与你我都有益处。我现在打发他亲自回你那里去；他是我心上的人。我本来有意将他留下，在我为福音所受的捆锁中替你伺候我。但不知道你的意思，我就不愿意这样行，叫你的善行不是出于勉强，乃是出于甘心。他暂时离开你，或者是叫你永远得着他，不再是奴仆，乃是高过奴仆，是亲爱的兄弟。在我实在是如此，何况在你呢！这也不拘是按肉体说，是按主说。<br/>

你若以我为同伴，就收纳他，如同收纳我一样。他若亏负你，或欠你什么，都归在我的帐上；我必偿还，这是我—<Pn>保罗</Pn>亲笔写的。我并不用对你说，连你自己也是亏欠于我。兄弟啊，望你使我在主里因你得快乐<Nt>或译：益处</Nt>，并望你使我的心在基督里得畅快。<br/>

我写信给你，深信你必顺服，知道你所要行的必过于我所说的。此外你还要给我预备住处；因为我盼望借着你们的祷告，必蒙恩到你们那里去。<br/>
